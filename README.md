# grab-wp-post
## A small web app that fetches posts from a WordPress site, displays them in cards and allow quick copying of informaion for fast and easy content reuse.

Live Demo
👉 [Grab WP Post](https://jppat.github.io/grab-wp-post "Visit Grab WP Post Live Demo")
**Built with Vite, React/JavaScript, HTML, and CSS**

I put this together because I needed a quicker way to pull content from a WordPress site I manage and share it on social media. I know there are tools out there that can do this, but they all charge recurring fees – something I can’t justify at the moment when I only use this one feature.

### Features

* Fetch WordPress posts via REST API
* Display posts in clean card UI
* One-click copy to clipboard
* Coming Soon! Integration with Facebook

## Usage

To use it, just provide the link to the WordPress website. You can also filter by category or date. Each post will be displayed in cards. Each card has a handy copy button that will copy all text displayed on the card.

## For Devs who want to play with the source code
1. `git clone https://github.com/Jppat/grab-wp-post.git`
2. `cd grab-wp-post`
3. `npm install`
4. `npm run dev`
